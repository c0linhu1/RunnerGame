A dynamic and endless runner game built with Unity featuring procedural level generation, challenging obstacles, and engaging gameplay mechanics.

## About

This game is an endless runner - similar to games Run, Run 2, etc. where players navigate through procedurally generated levels, avoiding obstacles and collecting points. This game features movement mechanics, increasing difficulty, as well as. a arcade-style gameplay.

## Features

- **Endless Runner Gameplay**: Procedurally generated levels for infinite replayability
- **Obstacle Avoidance**: Navigate through challenging obstacles and hazards
- **Score System**: Track your performance and compete for high scores
- **Smooth Controls**: Responsive player movement and jump mechanics
- **Progressive Difficulty**: Game gets more challenging as you progress

## Built With

- **Unity Engine** - Game development framework
- **C#** - Primary programming language

### Prerequisites

- [Unity Hub](https://unity.com/download)
- Unity Editor (compatible version will be specified in ProjectVersion.txt)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/c0linhu1/RunnerGame.git
```

2. Open Unity Hub

3. Click "Add" and navigate to the cloned directory

4. Select the project folder

5. Open the project in Unity Editor

### Running the Game

1. Open the project in Unity
2. Navigate to the main scene in the Scenes folder
3. Press the Play button in the Unity Editor to test the game
4. Use arrow keys or WASD to control the player
